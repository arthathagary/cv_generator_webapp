🔍 Extension Debug Helper

Step 1: Check Extension Background Script

1. Go to chrome://extensions/

2. Make sure "Developer mode" is ON

3. Find your "Resume AI" extension

4. Click "service worker" (or "background page") link

5. This opens the background script console

6. Look for any error messages in red

7. Try the connection test while watching this console

Step 2: Check Extension Storage

In the background script console, paste this command:

chrome.storage.sync.get(['webAppConnected', 'accessToken', 'userProfile']).then(console.log)

This will show you what's actually stored in the extension.

Step 3: Test Background Script Connection

In the background script console, paste this to test if it can receive connections:

console.log('Background script is running'); chrome.runtime.onConnect.addListener((port) => { console.log('TEST: Port connected:', port.name, 'from:', port.sender); port.onMessage.addListener((msg) => { console.log('TEST: Message received:', msg); }); });

Step 4: Check Extension Manifest

Make sure your extension has these permissions in manifest.json:

"externally_connectable": { "matches": [ "http://localhost:*/*", "https://localhost:*/*" ] }
